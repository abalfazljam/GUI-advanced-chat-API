## Overview

GUI Advanced Chat API is a lightweight, modern, and standalone AI chat interface built entirely with PHP.

It provides a clean and responsive user interface for communicating with OpenAI-compatible APIs while keeping the project simple, fast, and easy to deploy.

Unlike many AI chat applications that rely on complex frameworks or heavy dependencies, this project is designed to run with only a few PHP files, making it suitable for local development as well as shared hosting environments.

Its modular architecture allows developers to customize both the frontend and backend with minimal effort.

# Why GUI Advanced Chat API?

GUI Advanced Chat API was created to provide a clean, lightweight, and developer-friendly interface for interacting with Large Language Models.

Many existing AI chat interfaces require large frameworks, package managers, or complicated installation procedures. This project focuses on simplicity without sacrificing modern features.

Everything is organized into a small number of files, making the codebase easy to understand, modify, and extend.

Whether you are experimenting with AI models, building internal tools, or integrating your own OpenAI-compatible service, GUI Advanced Chat API provides a solid foundation while remaining lightweight and easy to maintain.
