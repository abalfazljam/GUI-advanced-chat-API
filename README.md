## معرفی

پروژه GUI Advanced Chat API یک رابط کاربری مدرن، واکنش گرا و سبک برای چت با مدل های زبانی بزرگ (LLM) است که کاملاً با PHP خالص نوشته شده و بدون نیاز به فریم ورک سنگین یا دیتابیس کار می کند.

این پروژه امکان اتصال به هر API سازگار با OpenAI (مثل OpenAI، Groq، OpenRouter، FreeLLMAPI محلی، Dahl و ...) را فراهم می‌کند و ویژگی‌های پیشرفته‌ای مانند **استریمینگ پاسخ‌ها**، **Markdown + KaTeX + Syntax Highlighting**، **مدیریت چند پرووایدر**، **آپلود فایل/تصویر/صوت/ویدیو**، **تاریخچه چت با سطل زباله**، **۱۰۰ تم رنگی**، **پشتیبانی دوزبانه (فارسی/انگلیسی)** و ... را ارائه می‌دهد.

طراحی شده برای:
- توسعه‌دهندگان و علاقه‌مندان به AI
- اجرا روی هاست‌های اشتراکی معمولی PHP
- استفاده محلی یا شخصی‌سازی آسان

## مهم‌ترین قابلیت‌ها

### رابط کاربری و تجربه کاربری
- رابط کاربری مدرن، مینیمال و کاملاً واکنش‌گرا (موبایل + دسکتاپ)
- پشتیبانی کامل RTL (فارسی) و LTR (انگلیسی) با تغییر زبان لحظه‌ای
- بیش از **۱۰۰ تم رنگی** جذاب
- انیمیشن‌های مختلف پاسخ (Typewriter، Word-by-word، Fade، Smooth یا بدون انیمیشن)
- اسکرول هوشمند (اگر بالا باشید مزاحم نمی‌شود)
- نمایش تدریجی پاسخ مدل + قابلیت **توقف تولید** در هر لحظه
- پیشنهادهای سریع (Custom Prompts قابل ویرایش – حداکثر ۴ عدد)
- ناوبری پیام‌های کاربر + جستجوی چت‌ها

### قابلیت‌های هوش مصنوعی
- پشتیبانی کامل از **APIهای سازگار با OpenAI** (OpenAI، Groq، OpenRouter، FreeLLMAPI، Dahl و هر سرویس مشابه)
- مدیریت چند پرووایدر (افزودن، ویرایش، تست اتصال، فعال‌سازی)
- انتخاب مدل (با جستجو) + حالت Auto
- **Streaming واقعی** (SSE) با fallback خودکار به non-streaming
- پشتیبانی از **جستجوی وب** (برای FreeLLMAPI)
- کنترل Temperature
- نمایش فرمول‌های ریاضی با **KaTeX**
- رندر Markdown کامل + **Syntax Highlighting** (Highlight.js)
- آپلود و ارسال فایل: تصویر (base64)، صوت، ویدیو، متن (txt/md/json/...) تا ۲۰ مگابایت
- نمایش توکن مصرفی هر چت

### مدیریت داده و شخصی‌سازی
- **بدون دیتابیس** – همه چیز در فایل‌های JSON ذخیره می‌شود
- تاریخچه چت کامل + جستجو + سطل زباله (بازیابی / حذف دائم)
- پروفایل کاربر و AI (نام + آواتار قابل آپلود)
- پشتیبان‌گیری و بازیابی چت‌ها (Export/Import JSON)
- ساختار ماژولار و ساده برای توسعه‌دهندگان

### فنی
- توسعه یافته با **PHP خالص** + Vanilla JS + CSS
- فایل‌های کم و ساختار ساده
- مناسب هاست‌های اشتراکی معمولی
- کدنویسی تمیز و قابل گسترش

## نیازمندی‌ها

- PHP 7.4 یا بالاتر (توصیه: 8.0+)
- افزونه **cURL** (ضروری برای ارتباط با APIها)
- افزونه **GD** (اختیاری – برای تغییر اندازه آواتارها)
- دسترسی نوشتن روی پوشه `data/` (به صورت خودکار ساخته می‌شود)
- وب‌سرور (Apache/Nginx) یا سرور محلی PHP (`php -S`)

## نصب و راه‌اندازی

### روش ۱: روی هاست اشتراکی
1. فایل‌های پروژه را دانلود یا کلون کنید.
2. همه فایل‌ها را در public_html (یا زیرپوشه) آپلود کنید.
3. مطمئن شوید پوشه `data/` قابل نوشتن باشد (معمولاً به صورت خودکار ساخته می‌شود با دسترسی 0777).
4. آدرس سایت را در مرورگر باز کنید.
5. به **تنظیمات** بروید و پرووایدر + کلید API را وارد کنید.
